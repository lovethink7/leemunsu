How It Works

이 사이트가 동작하는 방식

광고를 보기 위해 만든 빈 페이지가 아니라, 직접 사용 가능한 도구이면서 설명 문서 역할도 하도록 구성했습니다.

1. 모델 로드

페이지는 Teachable Machine에서 내보낸 `model.json`과 `metadata.json`을 불러와 브라우저에서 실행합니다. 별도 서버 추론이 아니라 사용자 브라우저에서 이미지 판별이 이뤄집니다.

2. 웹캠 프레임 입력

카메라 권한이 허용되면 웹캠 프레임이 연속으로 업데이트되고, 현재 화면의 손 모양이 각 클래스와 얼마나 비슷한지 확률로 표시됩니다.

3. 최고 확률 클래스로 판정

가위, 바위, 보 가운데 가장 높은 확률 값을 가진 클래스를 사용자의 현재 손으로 해석하고, 컴퓨터가 낸 랜덤 선택과 비교해 승패를 계산합니다.

4. 결과 해석

이 결과는 어디까지나 학습 데이터와 현재 프레임 상태의 영향을 받는 예측값입니다. 따라서 클래스 확률을 함께 보여주어 결과 해석 근거를 확인할 수 있도록 했습니다.

FAQ

자주 묻는 질문

실제 테스트에서 많이 생기는 질문을 정리했습니다.

왜 가위가 바위로 잘못 인식되나요?

손가락 사이 간격이 좁거나 측면 각도로 촬영되면 손 윤곽이 뭉개져 바위에 가깝게 보일 수 있습니다. 손을 정면으로 펴고 배경을 단순하게 맞추는 것이 도움이 됩니다.

모바일에서도 사용할 수 있나요?

브라우저가 카메라 접근과 TensorFlow.js 실행을 허용하면 사용할 수 있습니다. 다만 모바일은 화면 크기가 작아 손 전체가 프레임에 안정적으로 들어오도록 거리를 조절하는 편이 좋습니다.

개인정보는 어디로 전송되나요?

이 도구는 웹캠 프레임을 브라우저에서 즉시 처리하는 구조를 기본 전제로 사용합니다. 자세한 내용은 개인정보처리방침 페이지에 정리해 두었습니다.
